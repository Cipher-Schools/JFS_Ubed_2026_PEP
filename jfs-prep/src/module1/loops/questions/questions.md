# Loop Practice Questions

1. **Sum of Natural Numbers**: Write a program to calculate the sum of first N natural numbers using a while loop.
2. **Multiplication Table**: Ask the user for a number and print its multiplication table (up to 10).
3. **Factorial**: Calculate the factorial of a number using a for loop.
4. **Armstrong Number**: Check if a given number is an Armstrong number (e.g., 153 = 1^3 + 5^3 + 3^3).
5. **GCD of Two Numbers**: Find the Greatest Common Divisor of two numbers using loops.
