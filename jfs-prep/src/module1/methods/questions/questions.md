# Methods Practice Questions

1. **IsEven**: Write a method `boolean isEven(int n)` that returns true if number is even.
2. **Power Function**: Write a method `int power(int base, int exponent)` that calculates result without using Math.pow().
3. **Temperature Convert**: Write methods `celsiusToFahrenheit` and `fahrenheitToCelsius`.
4. **Prime Check**: Write a method `isPrime(int n)` and use it to print primes up to 100.
