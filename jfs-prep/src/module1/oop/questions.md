# OOP Practice Questions

1. **Bank Account**: Create a `BankAccount` class with deposit(), withdraw(), and checkBalance() - ensure balance cannot be negative.
2. **Library System**: Design `Book` and `Library` classes to add books and display available books.
3. **Method Overloading**: Create a `Calculator` class that adds 2 integers, 3 integers, or 2 doubles.
4. **Animal Sounds**: Create an abstract class `Animal` and subclasses `cat`, `dog`, `lion` with unique sounds.
5. **Interface**: Create `Playable` interface with `play()` method. Implement it in `Guitar` and `Piano` classes.
