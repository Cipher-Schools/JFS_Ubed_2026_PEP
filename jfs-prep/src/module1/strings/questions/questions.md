# String Practice Questions

1. **Count Vowels**: Count the number of vowels in a given string.
2. **Remove Spaces**: Remove all whitespace from a string.
3. **Anagram Check**: Check if two strings are anagrams (contain same characters).
4. **Word Count**: Count the number of words in a sentence.
5. **Character Frequency**: Print frequency of each character in a string (e.g., "apple" -> a:1, p:2, l:1, e:1).
