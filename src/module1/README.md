# Module 1: Core Java Foundations

Welcome to Module 1! This module covers the fundamental building blocks of Java programming.

## ðŸ“š Topics Covered

### 1. **Primitives** (`primitives/`)

Core data types in Java:

- `int`, `long`, `short`, `byte` - Integer numbers
- `float`, `double` - Decimal numbers
- `boolean` - True/false values
- `char` - Single characters

### 2. **Operators** (`operators/`)

Performing operations on data:

- Arithmetic: `+`, `-`, `*`, `/`, `%`
- Relational: `==`, `!=`, `<`, `>`, `<=`, `>=`
- Logical: `&&`, `||`, `!`
- Assignment: `=`, `+=`, `-=`, etc.

### 3. **Conditionals** (`conditionals/`)

Control flow based on conditions:

- `if-else` statements
- `switch-case` statements
- Ternary operator `? :`

### 4. **Loops** (`loops/`)

Repetitive execution:

- `for` loop - Known iterations
- `while` loop - Condition-based
- `do-while` loop - Execute at least once
- Enhanced `for-each` loop

### 5. **Arrays** (`arrays/`)

Storing multiple values:

- Single-dimensional arrays
- Multi-dimensional arrays
- Array manipulation and iteration

### 6. **Strings** (`strings/`)

Working with text:

- String creation and manipulation
- `StringBuilder` for efficient string building
- Common string methods

### 7. **Methods** (`methods/`)

Reusable code blocks:

- Method declaration and definition
- Parameters and return types
- Method overloading
- Recursion

### 8. **Generics** (`generics/`)

Type-safe code:

- Generic classes
- Generic methods
- Type parameters and bounds

### 9. **OOP (Object-Oriented Programming)** (`oop/`)

Core programming paradigm with subfolders:

#### `basics/`

- Classes and Objects
- Constructors
- Instance vs Static members

#### `encapsulation/`

- Access modifiers (`private`, `public`, `protected`)
- Getters and setters
- Data hiding

#### `inheritance/`

- Extending classes
- Constructor chaining
- `super` keyword
- Method overriding

#### `polymorphism/`

- Method overriding
- Dynamic method dispatch
- Runtime polymorphism

#### `abstraction/`

- Abstract classes
- Interfaces
- Implementation of contracts

#### `staticmembers/`

- Static variables
- Static methods
- Static blocks
