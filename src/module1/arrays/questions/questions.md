# Array Practice Questions

1. **Sum & Average**: Calculate sum and average of an array elements.
2. **Min & Max**: Find the smallest and largest element in an array.
3. **Reverse Array**: Reverse an array in-place (without using a second array).
4. **Count Even/Odd**: Count how many even and odd numbers are in an array.
5. **Search Element**: Check if a specific number exists in an array and print its index.
