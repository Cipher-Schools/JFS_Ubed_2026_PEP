# Conditional Logic Questions

1. **Leap Year Checker**: Write a program to check if a year is a leap year or not.
2. **Largest of Three**: Find the largest of three numbers without using arrays (use nested if-else).
3. **Vowel or Consonant**: Take a character input and check if it's a vowel or consonant using switch-case.
4. **Grade Calculator**: Input marks for 3 subjects, calculate average, and print grade (A/B/C/F).
5. **Day of Week**: Input a number (1-7) and print the day name (Monday-Sunday).
